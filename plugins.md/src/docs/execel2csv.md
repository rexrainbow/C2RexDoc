## Usage

1. download this 7z file, and unzip it.
2. edit `run.bat` file, the format is

```
excel2csv excel_filename
```

(put excel file in the same folder)

3. double click `run.bat`

Or, drag the excel file onto `excel2csv.exe` directly.
