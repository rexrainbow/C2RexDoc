Behavior : Replacer

Author:Rex.Rainbow

  • Replace instancne by fade-out itself, and create the target instance then fade-in it.
Download

Properties

Fade duration

  • Duration of fade-out or fade-in, in seconds.

Actions

Configure

Action:Set duration

  • Set the object's fade duration.
    • Duration ( number ) : Duration of fade-out or fade in, in seconds.

Replace

Action:Replace instance

  • Replace instance.
    • Target ( object ) : Target type of replacing instance.

Action:Replace instance to nickname type

  • Replace instance to nickname type.
    • Target ( string ) : Target type in nickname of replacing instance.

Expressions

UID

Expression:ReplacingInstUID

  • The UID of replacing instanc, return -1 if the replacing does not start.

Expression:ReplacingInstUID

  • The UID of replacing instanc, return -1 if the replacing does not start.

Conditions

Fade in

Condition:On fade-in started

  • Triggered when fade-out started

Condition:On fade-in finished

  • Triggered when fade-in finished

Condition:Is fade-in

  • Return true if instance is in fade-in stage

Fade out

Condition:On fade-out started

  • Triggered when fade-out started

Condition:On fade-out finished

  • Triggered when fade-out finished

Condition:Is fade-out

  • Return true if instance is in fade-out stage

Idle

Condition:Is idle

  • Return true if instance is in idle stage
