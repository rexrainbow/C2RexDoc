Behavior : 2D-Char-Array layout

Author:Rex.Rainbow

  • Map/create sprites by a 2d-char-array on layouter.
Download

Properties

Cell width

  • Cell width, in pixel.

Cell height

  • Cell height, in pixel.

Actions

Cell

Action:Set cell width

  • Set cell width.
    • Cell width ( number ) : Cell width, in pixel.

Action:Set cell height

  • Set cell height.
    • Cell height ( number ) : Cell height, in pixel.

Map

Action:Map content

  • Map content.
    • Object type ( object ) : Mapped object type.
    • Layer ( layer ) : Layer name of number.
    • Strings ( string ) : Strings for creating a 2d char-array, seprated by newline.

Expressions

Character

Expression:Char

  • Get mapped character.

Layout

Expression:InstUID

  • Get UID of layouted instance.

Expression:LX

  • Get logic X of layouted instance.

Expression:LY

  • Get logic Y of layouted instance.

Conditions

Character

Condition:On each character

  • Trigger by "Action:MapContent" to get mapped character.
