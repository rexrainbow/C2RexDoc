Plugin : Tilt alpha to left/right key

Author:Rex.Rainbow

  • Get left or right key event from tilt's alpha input.

Properties

Sensitivity

  • Sensitivity of tilt angle, in degree.

Actions

Sensitivity

Action:Set sensitivity angle

  • Set sensitivity angle of turning direction detection.
    • Sensitivity angle ( number ) : Sensitivity angle.

calibration

Action:Calibration

  • Set ZERO angle to current degree.

Expressions

Angle

Expression:ZEROAngle

  • Get ZERO angle from alpha direction.

Expression:RotateAngle

  • Get rotate angle from alpha direction.

Expression:SensitivityAngle

  • Get sensitivity angle of turning direction detection.

Conditions

Is down

Condition:Left arrow is down

  • Return true if Left arrow is currently held down by alpha direction.

Condition:Right arrow is down

  • Return true if Right arrow is currently held down by alpha direction.

Pressd

Condition:On Left arrow pressed

  • Triggered when Left arrow is pressed.

Condition:On Right arrow pressed

  • Triggered when Right arrow is pressed.

Condition:On any key pressed

  • Triggered when any arrow is pressed.

Released

Condition:On Left arrow released

  • Triggered when Left arrow is released by alpha direction.

Condition:On Right arrow released

  • Triggered when Right arrow is released by alpha direction.
