Download

Properties

Actions

Expressions

Orientation

Expression:Orientation

  • Get orientation value of window.

Expression:IsLandspcape

  • Retrun 1 if current view is landspcape.

Conditions

Orientation

Condition:On orientation changed

  • Triggered when orientation of window changed.

Condition:On changing to landspcape view

  • Triggered when changing to landspcape view .

Condition:On changing to portrait view

  • Triggered when changing to portrait view .

Condition:Is landspcape view

  • True if window is at landspcape view .

Condition:Is portrait view

  • True if window is at portrait view .
