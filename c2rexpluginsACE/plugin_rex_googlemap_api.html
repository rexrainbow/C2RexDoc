Plugin : Google map api

Author:Rex.Rainbow

  • Load google map api.
Download

Properties

Initial loading

  • Enable if you wish to load map at the start of the layout.
    • Options : ["No","Yes"]

Language code

  • Specifics language displayed in google map. Set "" to use local language.

API URL

  • API URL, set to "maps.google.cn" if user is in China.
    • Options : ["maps.googleapis.com","maps.google.cn"]

Actions

Language

Action:Set language code

  • Set displayed language. Set "" to use local language.
    • Language code ( string ) : Displayed language. Set "" to use local language.

Load

Action:Load

  • Load API.

URL

Action:Set api URL

  • Set api URL, set to "maps.google.cn" if user is in China.
    • Set API URL ( Combo ) : Set to "maps.google.cn" if user is in China.
      • Options : ["maps.googleapis.com","maps.google.cn"]

Expressions

Conditions

Load

Condition:On loaded

  • Triggered when api loaded.

Condition:Is loaded

  • Return true if api is loaded.

Condition:On error

  • Triggered when api loaded error.
