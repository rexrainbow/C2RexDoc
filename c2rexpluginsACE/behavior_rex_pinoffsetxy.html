Behavior : Pin offsetXY

Author:Rex.Rainbow

  • Stick to another object maintaining a relative offsetXY.
Download

Properties

Actions

Pin

Action:Pin to object with offset

  • Pin the object to another object with offset.
    • Pin to ( object ) : Choose the object to pin to.
    • Offset x ( number ) : Offset x in pixel
    • Offset y ( number ) : Offset y in pixel

Action:Pin to object

  • Pin the object to another object with current offset.
    • Pin to ( object ) : Choose the object to pin to.

Unpin

Action:Unpin

  • Unpin the object.

Expressions

Expression:PinnedUID

  • Get the UID of the object pinned to, or -1 if not pinned.

Conditions

Condition:Is pinned

  • Object is currently pinned to another object.
