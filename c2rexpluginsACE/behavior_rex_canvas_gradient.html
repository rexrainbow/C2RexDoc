Behavior : Gradient

Author:Rex.Rainbow

  • Fill gradient color at canvas.

Properties

Actions

Color stop

Action:Define color stop

  • Define color stop.
    • Offset ( number ) : Offset, from 0 to 1.
    • color ( string ) : Use color name, hex "#FFA500", "rgb(0-255,0-255,0-255)", "rgba(0-255,0-255,0-255,0-1)", "hsl(0-360,0-100%,0-100%)", or "hsla(0-360,0-100%,0-100%,0-1)"

Draw

Action:Draw gradient

  • Draw gradient.

Filled area

Action:Define by center

  • Define filled area.
    • Center X ( number ) : X co-ordinate of center point.
    • Center Y ( number ) : Y co-ordinate of center point.
    • Width ( number ) : Width.
    • Height ( number ) : Height.

Action:Define by left-top

  • Define filled area.
    • Left-top X ( number ) : X co-ordinate of left-top point.
    • Left-top Y ( number ) : Y co-ordinate of left-top point.
    • Width ( number ) : Width.
    • Height ( number ) : Height.

Gradient - line

Action:Define line gradient

  • Define line gradient.
    • Start X ( number ) : X co-ordinate of start point.
    • Start Y ( number ) : Y co-ordinate of start point.
    • End X ( number ) : X co-ordinate of end point.
    • End Y ( number ) : Y co-ordinate of end point.

Gradient - radius

Action:Define radius gradient

  • Define radius gradient.
    • Start X ( number ) : X co-ordinate of start circle.
    • Start Y ( number ) : Y co-ordinate of start circle.
    • start radius ( number ) : Radius of start circle.
    • End X ( number ) : X co-ordinate of end circle.
    • End Y ( number ) : Y co-ordinate of end circle.
    • End radius ( number ) : Radius of end circle.

Action:Define concentric circle gradient

  • Define concentric circle gradient.
    • X ( number ) : X co-ordinate of concentric circle.
    • Y ( number ) : Y co-ordinate of concentric circle.
    • start radius ( number ) : Radius of start circle.
    • End radius ( number ) : Radius of end circle.
