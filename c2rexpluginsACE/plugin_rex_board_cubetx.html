Plugin : Cube Tx

Author:Rex.Rainbow

  • Transfer logic position to physical position with 3D orthogonal or isometric layout
Download

Properties

Orientation

  • Map orientation.
    • Options : ["Orthogonal","Isometric"]

X at (0,0,0)

  • Physical X co-ordinate at logic (0,0,0).

Y at (0,0,0)

  • Physical Y co-ordinate at logic (0,0,0).

Width

  • Cell width in pixels.

Height

  • Cell height in pixels.

Deep

  • Cell deep in pixels.

Actions

Orientation

Action:Set orientation

  • Set orientation.
    • Orientation ( Combo ) : orientation of map
      • Options : ["Orthogonal","Isometric"]

Action:Set orientation by number

  • Set orientation.
    • Isometric ( number ) : 1=Isometric, 0=Orthogonal

Position

Action:Set position offset

  • Set Physical position offset (position of logic (0,0)).
    • X ( number ) : Physical X co-ordinate at logic (0,0).
    • Y ( number ) : Physical Y co-ordinate at logic (0,0).

Size

Action:Set cell size

  • Set cell size.
    • Width ( number ) : Cell width in pixels.
    • Height ( number ) : Cell height in pixels.
    • Deep ( number ) : Cell deep in pixels.
