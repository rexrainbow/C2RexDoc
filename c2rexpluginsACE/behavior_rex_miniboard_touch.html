Behavior : (Miniboard) Touch Ctrl

Author:Rex.Rainbow

  • Drag & drop mini board.
Download

Properties

Enable

  • Enable to drag mini board.
    • Options : ["No","Yes"]

Align to grids

  • Enable to align mini board to grids.
    • Options : ["No","Yes"]

Actions

Align

Action:Set align mode

  • Set align mode of this miniboard.
    • Align ( Combo ) : Enable to align this miniboard to grids.
      • Options : ["No","Yes"]

Drag

Action:Try drag

  • Try to drag this object if is in touched.

Drag & Drop

Action:Set enable

  • Set drag-able of this miniboard.
    • Enable ( Combo ) : Enable to drag this miniboard.
      • Options : ["No","Yes"]

Drop

Action:Force to drop

  • Force the dragged object to drop.

Expressions

Main board

Expression:LX

  • Get logical X overlapped main board. Return -1 if out of any main board.

Expression:LY

  • Get logical Y overlapped main board. Return -1 if out of any main board.

Expression:MBUID

  • Get UID of overlapped main board. Return -1 if out of any main board.

Start

Expression:StartX

  • Get X co-ordinate of object's dragging start position.

Expression:StartY

  • Get Y co-ordinate of object's dragging start position.

Expression:DragStartX

  • Get X co-ordinate of dragging start position.

Expression:DragStartY

  • Get Y co-ordinate of dragging start position.

Conditions

Drag

Condition:On dragging start

  • Triggered when miniboard dragging start.

Condition:On logical position changed

  • Triggered when logical position changed on main board.
    • Main board ( object ) : Main board object.

Condition:Is drag-able

  • Return true if this mini board is drag-able.

Condition:On dragging at main board

  • Triggered when miniboard dragging at main board.
    • Main board ( object ) : Main board object.

Drop

Condition:On dropped

  • Triggered when miniboard dropped.

Condition:On dropped at main board

  • Triggered when dropped at main board.
    • Main board ( object ) : Main board object.

Touch

Condition:Is touching

  • Return true if this mini board is touching.
