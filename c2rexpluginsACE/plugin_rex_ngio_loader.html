Plugin : Loader

Author:Rex.Rainbow

  • Handles loading various URLs and tracking referral stats.
Download

Properties

Host

  • The domain hosting your app.

Actions

Newgrounds

Action:Load Newground page

  • Loads Newgrounds, and logs the referral to your API stats.
    • Operation ( Combo ) : Operation.
      • Options : ["Get url of","Open"]

Action:Load more game

  • Loads the Newgrounds game portal, and logs the referral to your API stats.
    • Operation ( Combo ) : Operation.
      • Options : ["Get url of","Open"]

Official URLs

Action:Load author page

  • Loads the official URL of the app's author (as defined in your "Official URLs" settings), and logs a referral to your API stats.
    • Operation ( Combo ) : Operation.
      • Options : ["Get url of","Open"]

Action:Load official page

  • Loads the official URL where the latest version of your app can be found (as defined in your "Official URLs" settings), and logs a referral to your API stats.
    • Operation ( Combo ) : Operation.
      • Options : ["Get url of","Open"]

Referral

Action:Load referral

  • Loads a custom referral URL (as defined in your "Referrals & Events" settings), and logs the referral to your API stats.
    • Operation ( Combo ) : Operation.
      • Options : ["Get url of","Open"]
    • Referral ( string ) : The name of the referral (as defined in your "Referrals & Events" settings).

Expressions

Result

Expression:ErrorMessage

  • Get last error message from last result.

Expression:LastURL

  • Get url from last result.

Conditions

Get URL

Condition:On get url

  • Triggered when get url success.

Condition:On get url error

  • Triggered when get url error.
