Behavior : Linear layout

Author:Rex.Rainbow

  • Put instances linear on layouter.
Download

Properties

Mode

  • Average mode: layout instances in range averagely, Fix mode: layout instances with fix distance.
    • Options : ["Average","Fix"]

Direction

  • Direction from start to end.
    • Options : ["Left to right","Right to left","Top to bottom","Bottom to top"]

Alignment

  • Alignment. It only uses in Fix mode.
    • Options : ["Start","Center","End"]

Spacing

  • Spacing (delta distance) of each instance, in pixels. It only uses in Fix mode.
