Plugin : ToneJS api

Author:Rex.Rainbow

  • Api of tone.js. https://github.com/Tonejs/Tone.js

Properties

Transport

  • Transport for timing musical events.

Start timeline

  • Set Yes to start timeline.
    • Options : ["No","Yes"]

BPM

  • The Beats Per Minute of the Transport.

Actions

Property

Action:Set value

  • Set property of an object.
    • Object ( object ) : Object to assign.
    • Property ( string ) : Property name in dot notation
    • Value ( number or string ) : Value to set

Action:Set JSON

  • Set property of an object.
    • Object ( object ) : Object to assign.
    • Property ( string ) : Property name in dot notation
    • JSON ( string ) : JSON value to set

Action:Set boolean

  • Set property of an object.
    • Object ( object ) : Object to assign.
    • Property ( string ) : Property name in dot notation
    • Boolean ( Combo ) : Boolean value.
      • Options : ["false","true"]

Expressions

Property

Expression:Property

  • Get property of an instance.
    • UID ( number ) : UID of a tone instance
    • Property ( string ) : Property name in dot notation

Time

Expression:Now

  • Get the currentTime from the AudioContext.
